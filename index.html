Come installare

1

Scarica il file .exe qui sopra e aprilo — Windows potrebbe mostrare un avviso SmartScreen: clicca «Altre informazioni» → «Esegui comunque»

2

L'installer ti chiede la cartella di destinazione, poi copia i file e crea un collegamento sul desktop — finisce in pochi secondi

3

Avvia Strip Board IVAO dal desktop; l'app si apre nel browser di default e ascolta sulla porta 3000

4

Dal tablet (stessa rete WiFi) apri http://<IP-del-PC>:3000 — trova l'IP in Aurora o col comando ipconfig

5

Gli aggiornamenti futuri arrivano automaticamente: l'app ti avvisa e installa con un clic, senza doverla riscaricare

Requisiti

SistemaWindows 10 o superiore (64-bit)
Aurora3rd Party Software Access = YES in F7 → Other
RetePC e tablet sulla stessa rete WiFi per l'accesso tablet
Account IVAONecessario per i dati Whazzup (chiave API configurata nel build)
DipendenzeNessuna — Node.js e il server sono inclusi nell'installer
